# My experience with the K&R C book
## Chapter 1:

### Notes
#### Data Types
- data types like `int` and `float`: can be 16-bit, 32-bit, etc, depending on architecture.
- `char`: 8-bit data type representing ascii characters.
- `short`: short int.
- `long`: long int.
- `float`: typically 32-bit, with at least 6 significant digits.
- `double`: double-precision floating point.

#### printf arguments
(also apparently printf is not part of C.)
- `%d`: int argument.
- `%3d`: int argument in a 3 digit-wide field.
- `%f`: float argument.


#### special characters
- `\t`: tab
- `\n`: newline
- `\"`: escaped double-quote
- `\b`: backspace
- `\\`: escaped backslash
